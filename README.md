# 🛒 AI Shopping Assistant

An AI-powered e-commerce chatbot that recommends products using NLP techniques and a modern interactive UI.

---

## 🚀 Features

- 🔍 Smart product search using natural language queries  
- 💰 Price-based filtering (e.g., "under 20000")  
- 🧠 AI recommendations using TF-IDF and cosine similarity  
- 🛍️ Modern card-based UI with grid layout  
- ⚡ Fast and interactive Streamlit app  

---

## 🧠 Tech Stack

- Python  
- Streamlit  
- Pandas  
- Scikit-learn (TF-IDF, Cosine Similarity)  

---

## 💡 How It Works

1. User enters a query (e.g., "best phone under 20000")  
2. System extracts price and intent  
3. Converts text into TF-IDF vectors  
4. Finds similar products using cosine similarity  
5. Displays top recommendations
